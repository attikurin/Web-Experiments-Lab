# 🧪 Web Experiments Lab

知的好奇心を刺激する実験的Webアプリケーション集

## 📋 プロジェクト概要

![Web Experiments Lab](images/site-thumbnail.jpg)

Web Experiments Labは、最新のWeb技術を活用した実験的プロジェクトのショーケースサイトです。AR、3Dグラフィックス、ゲーム、シミュレーションなど、多様なジャンルのWebアプリケーションを紹介しています。

**ALL GAMES POWERED BY GEMINI** - すべてのゲームはGemini AIの支援により開発されています。

### コンセプト
- **知的**: 技術的な探求心を刺激
- **洗練**: モダンでスタイリッシュなデザイン
- **最新**: 最新のWeb技術を採用
- **遊び**: インタラクティブで楽しい体験
- **ジョーク**: ユーモアとウィットを随所に

## ✨ 完成済み機能

### 現在公開中のプロジェクト

1. **AR Art Studio**
   - 拡張現実技術を活用したデジタルアートスタジオ
   - 現実空間にバーチャル作品を配置
   - URL: https://attikurin.github.io/AR-Art-Studio/

2. **3D Minesweeper**
   - 3次元空間のマインスイーパー
   - 立体的思考が求められる新感覚パズル
   - URL: https://attikurin.github.io/3DMinesweeper/

3. **ダメ社員マネージャー**
   - 残念な社員を管理するシミュレーションゲーム
   - ブラックユーモア満載のマネジメント体験
   - URL: https://attikurin.github.io/bad-employee-manager/

4. **大先生バトル**
   - 戦略バトルゲーム
   - ターン制で繰り広げられる知的バトル
   - URL: https://attikurin.github.io/daisenseibattle/

5. **大先生討伐戦**
   - 協力型レイドバトル
   - エピックなボス戦体験
   - URL: https://attikurin.github.io/daisensei-Subjugation/

6. **Cosmic Particles**
   - Webカメラで光を操るインタラクティブアート
   - MediaPipeハンドトラッキング＋Three.jsパーティクルシステム
   - URL: https://attikurin.github.io/Cosmic-Particles/

7. **全自動・顔モザイクメーカー**
   - プライバシー保護に特化した顔モザイク加工アプリ
   - face-api.js（SSD MobileNet V1）で群衆・横顔・小さな顔も高精度検出
   - 手動補正モード（ナンバープレート等）、動画対応（Beta）
   - 完全クライアントサイド処理で安心（サーバー送信なし）
   - URL: https://attikurin.github.io/FaceMosaicApp/

8. **クイック席替えメーカー**
   - 30秒で座席表が完成する究極にシンプルなアプリ
   - Excelからコピペで超高速入力、柔軟なレイアウト設定（列×行）
   - クリックで席を入れ替えて微調整、印刷・PDF最適化（A4横）
   - URL: https://attikurin.github.io/Quick-Seat-Change-Maker/

9. **教室用ビジュアルタイマー**
   - 子どもにも分かりやすい教室用タイマー
   - 円グラフで残り時間可視化、色の変化（青→黄→赤）
   - 複数タイマー同時管理、全画面モード、選べる通知音（チャイム・読み上げ等）
   - URL: https://attikurin.github.io/Classroom-Timer/

10. **クラスメモボード**
    - クラスの情報を一箇所に集約するデジタル掲示板
    - 今日の予定、提出物チェックリスト、当番表、簡易メモ機能
    - 自動保存、印刷・PDF保存対応、黒板・ホワイトボードの代替に最適
    - URL: https://attikurin.github.io/class-memo-board/

11. **ランダムセレクター**
    - 楽しくて公平な抽選・グループ分けアプリ
    - ルーレットモード（アニメーション付き）、グループ分けモード（2〜8チーム）
    - 除外機能、大画面モード、紙吹雪エフェクト搭載
    - URL: https://attikurin.github.io/RandomSelector/

12. **FormMirror**
    - 動作分析・フォームチェックアプリ
    - お手本動画と自分の動きを2画面同時再生比較、MediaPipe AI骨格検出
    - 体育授業・部活動・スポーツトレーニングに最適
    - URL: https://attikurin.github.io/FormMirror/

13. **万能PNGコンバーター**
    - PDF、画像、Office（Excel・Word・PowerPoint）、テキストをPNG画像に変換
    - 完全クライアントサイド処理、一括ZIP保存対応
    - PDFは全ページ高解像度分割、Officeファイルも専用ソフト不要で画像化
    - URL: https://attikurin.github.io/Universal-PNG-Converter/

14. **スマートグループ分け**
    - 参加者を自動で最適なチームに分割
    - 男女比均等化、役割自動割り当て対応
    - URL: https://attikurin.github.io/GroupMaker/

## 🚀 デプロイ方法

### GitHub Pages
このプロジェクトは静的サイトなので、GitHub Pagesで簡単にデプロイできます。

1. GitHubリポジトリを作成
2. ファイルをプッシュ
3. Settings → Pages → Source を選択
4. `main`ブランチの`/root`を選択

### その他のホスティング
- **Netlify**: ドラッグ&ドロップでデプロイ
- **Vercel**: Git連携で自動デプロイ
- **Cloudflare Pages**: 高速CDN配信
